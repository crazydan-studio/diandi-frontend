前端开发
===================================

## 工程构建

安装依赖：

```bash
yarn install
```

> `npm install`会安装失败，只能通过`Yarn`安装依赖。

启动本地调试：

```bash
npm run dev
```

构建发布产物：

```bash
npm run prod
```
